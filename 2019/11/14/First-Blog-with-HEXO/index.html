First Blog with HEXO

第一次搭建基于HEXO的Blog

作为一个工科生，一直对github不甚了解。也因为一直忙于科研学业，未顾及记录及分享自己的一些体会。前阵子稍微闲了下来，想找个博客用于记录自己的一些小项目和想法，发现了居然github可以当博客用。费劲一番周折，终于开通了这个Blog。希望写Blog可以督促自己在不断学习的过程中能够反思和记录。Let’s begin!

PS:这篇Blog是使用VSCODE写的，HEXO是通过WIN10的WSL子系统安装在Ubuntu里面的，写完了发现VSCODE居然保存不了。对于几乎没有接触过Linux的我来说，只知道Linux的权限分配很强，没想到目录的读写权限也这么严格。修改文件夹权限为777最终解决。 

chmod 777 -R foldername
